QUIZ 3 (HTML)

Para criar listas sem numeração (apenas com marcadores), o elemento HTML utilizado é o <ul>...</ul>. Cada item da lista deve ser um elemento <li>...</li>.
Para criar listas numeradas, o elemento HTML utilizado é o <ol>...</ol>. Cada item da lista deve ser um elemento <li>...</li>.
Considerando que um elemento não pode ter a tag de fechamento fora do elemento onde está a sua tag de abertura, qual o código mais correto para gerar a lista abaixo:

                                <h1>Minhas comidas preferidas</h1>
                                <ul>
                                    <li>Arroz

                                    </li>
                                    <li>Queijo
                                        <ol><li>Mozzarella<li>Flamengo</ol>
                                    </li>
                                </ul>
